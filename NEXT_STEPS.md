## Technical Debt / Known Issues

1. **Payroll contract does not transfer real XLM.** Currently records claim event only. If time permits, add `token.transfer` using Soroban token contract.
2. **Freighter wallet integration is TODO.** Claim submission currently shows a placeholder message. Integrate `@stellar/freighter-api` for browser signing.
3. **Browser zkey loading.** `paycheck_final.zkey` may be too large for browser. Node.js fallback (`scripts/generate-proof.js`) is already implemented.
4. **circomlibjs vs Circom Poseidon mismatch.** Merkle tree in JS must use Circom's compiled WASM witness calculator, not circomlibjs's Poseidon. Already handled in `merkle.ts`.
5. **Public input order.** snarkjs public signals follow witness signal declaration order, NOT Circom's `{public [...]}` declaration order. Current order: `amount, root, nullifier`.
